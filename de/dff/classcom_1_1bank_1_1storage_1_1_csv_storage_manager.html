Detailed Description

Provides CSV-based persistence for Storable objects.

The CsvStorageManager class implements the StorageManager interface and allows reading and writing data in CSV format to and from files.

It supports both single-object and multi-object storage, with optional append mode. The class is typically used for saving entities like accounts, transactions, or users in a human-readable format.

Usage Example:

CsvStorageManager manager = new CsvStorageManager();
manager.saveAll(listOfAccounts, "accounts.csv", false);
