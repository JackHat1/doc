Detailed Description

Represents a withdrawal transaction from a bank account.

The Withdrawal class extends Transaction and handles the process of withdrawing a specified amount of money from an account. If sufficient balance exists, it deducts the amount and records a StatementEntry of type Debit.

Execution Flow:

  1. Check that the account’s balance is sufficient.
  2. Withdraw the requested amount from the account.
  3. Record the operation in the user’s statement using StatementManager.
Note
If the balance is insufficient, the withdrawal is aborted and a message is printed to the console.

◆ execute()

void com.bank.model.transactions.Withdrawal.execute ( )

Executes the withdrawal transaction.

If sufficient funds are available:

  • Deducts the amount from the account balance.
  • Prints a confirmation message.
  • Creates a StatementEntry of type Debit and stores it via StatementManager#saveStatement(Account, StatementEntry).

Otherwise, prints a message indicating insufficient balance.
