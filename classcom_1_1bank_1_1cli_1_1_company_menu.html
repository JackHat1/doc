Detailed Description

CLI interface for company users to manage their accounts and issued bills.

Provides options for a company-type user to:

  • View all owned business accounts
  • Create and issue new bills (RF code + amount + customer VAT)
  • View all issued bills
  • View paid bills

Dependencies:

Uses AccountManager for account lookups and BillManager for bill creation and persistence.
