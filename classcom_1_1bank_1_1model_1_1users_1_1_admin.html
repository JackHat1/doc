Detailed Description

Represents an administrative user in the banking system.

The Admin class extends the User base class and defines privileged users who can manage other users, accounts, transactions, and system-wide configurations.

Key Characteristics:

  • Role is fixed as "Admin".
  • Can perform higher-level operations not allowed for regular users.
  • Implements serialization methods marshal() and unmarshal(String).
