Detailed Description

CLI menu for individual users: accounts, deposits, withdrawals, transfers, bill payments, and statements.

Provides an interactive loop for an User with role "Individual" to:

  • View own accounts and balances
  • Make deposits and withdrawals
  • Transfer money to another IBAN
  • Pay a bill by RF code
  • View account statements

Dependencies:

Uses AccountManager, TransactionManager, BillManager, StatementManager and a shared Scanner for console input.
