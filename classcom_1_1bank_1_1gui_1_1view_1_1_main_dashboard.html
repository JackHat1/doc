Detailed Description

Main Swing frame acting as role-aware dashboard (Individual / Company / Admin).

Renders a left-side menu and a central content area controlled by a CardLayout. Menu items are generated dynamically based on the logged-in user’s role and each item maps to a corresponding JPanel screen (Accounts, Transactions, Bills, Admin tools, etc.).

High-level flow:

  1. Construct managers required by the dashboard panels (e.g., BillManager, TransactionManager).
  2. Build header (logo + title), left menu, and center content panel with CardLayout.
  3. For each menu entry, create the corresponding panel and register a button handler to show it.
  4. On "Logout", dispose the frame; the app caller is expected to relaunch the LoginWindow.
