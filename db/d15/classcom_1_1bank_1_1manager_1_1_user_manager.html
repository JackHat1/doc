Detailed Description

Handles loading, saving, and managing all user accounts.

The UserManager is responsible for managing all users in the banking system, including Individual, Company, and Admin accounts.

It supports:

  • Loading users from a CSV file.
  • Saving all user data back to storage.
  • Adding new users with duplicate prevention.
  • Finding users by username or VAT number.

Storage format:
Each user is serialized using the User#marshal() method and stored in ./data/users/users.csv.

◆ load()

void com.bank.manager.UserManager.load ( )

Loads all users from the CSV file.

Reads user records line-by-line and creates User objects according to the “type” field:

  • Individual → Individual
  • Company → Company
  • Admin → Admin

If a line contains missing or invalid fields, it is skipped.
