Detailed Description

add_doxygen_stubs.py
--------------------
Βοηθητικό εργαλείο για να προσθέσει γρήγορα Doxygen/Javadoc stubs
σε projects με πολλές κλάσεις (C/C++/Java).

Χρήση:
    python tools/add_doxygen_stubs.py --roots src include --ext .h .hpp .c .cpp .java

Τι κάνει:
- Σαρώνει αρχεία και αν δεν βρίσκει Doxygen block πριν από κλάσεις/συναρτήσεις,
  εισάγει ένα ελάχιστο πρότυπο.
- *Ασφαλές*: κρατά backup του αρχείου(.bak) πριν την αλλαγή.
Σημ.: Είναι απλό regex-based εργαλείο, δεν καταλαβαίνει όλες τις γλωσσικές γωνίες.
