Detailed Description

Απλό Swing παράθυρο σύνδεσης (username/password) για το e-banking GUI.

Προβάλει λογότυπο/τίτλο, φόρμα με πεδία username & password και ένα κουμπί "Login". Η αυθεντικοποίηση δεν γίνεται εδώ — η κλάση εκπέμπει το event μέσω του LoginListener και ο caller (π.χ. EBankingApp) χειρίζεται τον έλεγχο.

Χρήση:

LoginWindow login = new LoginWindow();
login.setLoginListener((u, p) -> {
    // έλεγχος διαπιστευτηρίων και πλοήγηση
});
