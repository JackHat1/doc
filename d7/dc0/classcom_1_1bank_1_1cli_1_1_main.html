Detailed Description

CLI entry point for the banking application.

Bootstraps managers, loads persisted data, performs interactive login and routes the logged-in user to the appropriate role-specific menu:

  • Individual → IndividualMenu
  • Company → CompanyMenu
  • Admin → AdminMenu

Lifecycle:

  1. Load users, accounts, and bills from storage.
  2. Prompt for credentials and authenticate.
  3. Dispatch to role menu and run the CLI loop.
  4. On exit, persist updated users & accounts.
Note
Τα IndividualMenu, CompanyMenu, AdminMenu θεωρούνται διαθέσιμα στο ίδιο project (παρέχουν τα αντίστοιχα CLI flows).
