Detailed Description

Swing entry point for the e-banking graphical client.

Bootstraps the application on the Swing Event Dispatch Thread (EDT), creates and loads core managers (UserManager, AccountManager, BillManager), and presents the login window. On successful login, opens the MainDashboard.

Lifecycle:

  1. Run on EDT via SwingUtilities#invokeLater(Runnable).
  2. Instantiate managers and load data from ./data folders.
  3. Show LoginWindow and authenticate.
  4. On success: open MainDashboard. On dashboard close: return to login.

Error handling: If data loading fails, shows an error dialog and exits.
