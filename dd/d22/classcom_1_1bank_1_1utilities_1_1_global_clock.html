Detailed Description

A global time utility class used to simulate and manipulate the system date and time.

The GlobalClock class provides a central point of reference for all date and time operations within the banking system. It allows the simulation of time progression, which is especially useful for testing scheduled operations such as standing orders or interest calculations.

Usage Example:

GlobalClock.addDate(0, 1, 0); // Move simulated time forward by one month
LocalDate today = GlobalClock.getDate();
