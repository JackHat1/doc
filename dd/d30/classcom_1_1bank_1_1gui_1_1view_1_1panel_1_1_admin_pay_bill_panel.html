Detailed Description

Επιτρέπει σε διαχειριστή να πληρώσει λογαριασμό (Bill) εκ μέρους πελάτη.

Ροή χρήσης:

  1. Εισαγωγή VAT πελάτη και "Load Accounts" για φόρτωση λογαριασμών του.
  2. Επιλογή λογαριασμού χρέωσης (IBAN) από combo box.
  3. Εισαγωγή RF Code του λογαριασμού προς πληρωμή και "Pay".
  4. Γίνονται έλεγχοι ύπαρξης πελάτη, υπολοίπου, ύπαρξης/κατάστασης λογαριασμού κ.λπ.

Χρησιμοποιεί:

  • UserManager για αναζήτηση πελάτη με VAT.
  • AccountManager για ανάκτηση λογαριασμών πελάτη.
  • BillManager για εύρεση Bill από RF code.
  • TransactionManager για εκτέλεση Payment.
