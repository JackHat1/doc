Detailed Description

Defines the serialization and deserialization behavior for persistent objects.

The Storable interface provides a standard contract for converting objects to and from a textual representation, typically for file storage (e.g., CSV).

Classes implementing this interface can be easily saved or loaded by storage managers such as CsvStorageManager.

Usage Example:

public class Account implements Storable {
private String iban;
private double balance;
@Override
public String marshal() {
return iban + "," + balance;
}
@Override
public void unmarshal(String data) {
String[] parts = data.split(",");
iban = parts[0];
balance = Double.parseDouble(parts[1]);
}
}
