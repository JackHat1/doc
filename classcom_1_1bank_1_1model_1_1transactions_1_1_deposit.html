Detailed Description

Represents a deposit transaction to a single bank account.

A Deposit is a concrete subclass of Transaction that increases an account’s balance by a specified amount. Upon execution, it:

  1. Calls Account#deposit(double) to update the balance.
  2. Creates a StatementEntry describing the deposit.
  3. Saves the statement entry using StatementManager#saveStatement(Account, StatementEntry).
Note
The deposit amount is assumed to be positive; no validation is performed here. The transaction type in the statement is recorded as "Credit".
